# Azure Data Engineer Roadmap

This repo tracks my practical journey from Python/data foundations to Azure Data Engineer job readiness by December 2026.

## Target Role

Azure Data Engineer / Analytics Engineer / Python Data Platform Engineer

## Main Goals

- Build strong Python, SQL, Git, Docker, and data modelling foundations.
- Prepare for Microsoft DP-700 Fabric Data Engineer Associate.
- Prepare for Databricks Certified Data Engineer Associate.
- Build portfolio projects that prove real cloud data engineering skill.
- Become ready for U.S.-style technical interviews and recruiter review.

## Weekly Study System

- 10 hours deep study: Microsoft Learn, docs, notebooks, exam objectives.
- 8 hours lab building: code, deploy, debug, test, document.
- 3 hours review: notes, practice questions, weak-topic repair.
- 2 hours career output: README updates, LinkedIn posts, resume bullets, mock interviews.

## Roadmap

| Phase | Focus | Outcome |
|---|---|---|
| Phase 0 | Python, SQL, Git, Docker, data modelling | Strong technical foundation |
| Phase 1 | Microsoft Fabric / DP-700 | Fabric data engineering skill |
| Phase 2 | Databricks, PySpark, Delta Lake | Lakehouse engineering skill |
| Phase 3 | Azure backend/cloud integration | FastAPI, containers, queues, monitoring |
| Phase 4 | Portfolio and job readiness | Resume, LinkedIn, GitHub proof, interviews |
